# Tmux

change prefix from `Ctrl + b` to `Ctrl + a`

`Ctrl` + `a` - `c`: create window

`Ctrl` + `a` - `,`: rename window

`Ctrl` + `a` - `n`: go to next window

`Ctrl` + `a` - `p`: go to previous window

`Ctrl` + `a` - `w`: see all windows

`Ctrl` + `a` - `I`: install plugins
